# Task 4 - Online Examination System (Java Swing)

## Project Description
This project is a Java Swing based Online Examination System that provides a GUI interface for users to login, update profile and password, attempt MCQ based exam, and auto-submit when the timer ends.

## Features
- Login System with limited attempts
- Update Profile (Name)
- Update Password
- MCQ Based Examination
- Timer Countdown
- Auto Submit when time is over
- Score Calculation
- Logout and Session Closing

## Technologies Used
- Java
- Java Swing (JFrame, JOptionPane)
- Object-Oriented Programming (OOP)
- Timer functionality
