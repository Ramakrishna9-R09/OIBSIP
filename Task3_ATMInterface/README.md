# Task 3 - ATM Interface (Java)

## Project Description
This is a console-based ATM Interface project developed using Java.  
The project simulates basic ATM operations such as login, withdraw, deposit, transfer, and transaction history.

## Features
- User Login using User ID and PIN
- Withdraw Money with balance validation
- Deposit Money
- Transfer Money to another account
- Transaction History
- Daily withdrawal limit
- File saving of transaction history
- Logout/Quit option

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Collections Framework (HashMap, ArrayList)
- File Handling
- Date and Time API

## How to Run
1. Download or clone the repository
2. Open the project in VS Code / IntelliJ
3. Compile and run:

```bash
javac ATMProject.java
java ATMProject
