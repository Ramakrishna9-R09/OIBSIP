# Task 2 - Guess The Number Game (Java)

## Project Description
This is a Java console-based game where the system generates a random number between 1 and 100.  
The player must guess the number within a limited number of attempts.

## Features
- Random number generation (1 to 100)
- User input based guessing
- Feedback for each guess (Too High / Too Low)
- Maximum attempts limit (10 attempts per round)
- Multiple rounds option
- Score calculation based on remaining attempts
- Final score display

## Technologies Used
- Java
- Loops and Conditions
- Random Class
- Scanner for input handling

## How to Run
1. Download or clone the repository
2. Open the project in VS Code / IntelliJ
3. Compile and run:

```bash
javac GuessTheNumber.java
java GuessTheNumber
Output

The program displays whether the guess is too high or too low and calculates score.
